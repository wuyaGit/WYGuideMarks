## 特点：
1. '我知道'按钮位置定义
2. 直接跳过引导
3. 自定义指示箭头是否显示
4. 自定义要显示的属性

## 三种操作方式：
1. 点击背景，跳到下一步
2. 点击镂空，跳到下一步
3. 点击'我知道'，跳到下一步
